0x00 整数类型

1.有符号整数（int） 长度为4字节 可表示数值范围0到2^32-1 16进制范围为：0x00000000~0xFFFFFFFF 十进制表示为：0~4294967295

2.无符号整数（unsigned int） 长度为4字节 可表示数值范围-2^32到~2^32-1 -2147483648~2147483647 16进制为0x80000000~0x7FFFFFFF 0x80000000对应的二进制为 1000 0000 0000 0000 0000 0000 0000 0000 负数的存储采用了补码的形式，现代计算机中用补码表示负数

0x01 浮点数类型

浮点数类型的存储方式可以分为两种： 1、定点实数存储方式 2、浮点实数存储方式 对于现代计算机，采用的是浮点实数存储方式 在 C++ 中，有两种表示浮点数的方式，“float”用4字节表示浮点数，“double”用8字节表示浮点数。 float类型在内存中占4字节，最高位表示符号位，剩下31位，从右向左取8位用于表示指数，其余用于表示尾数

在进行二进制转化前，需要进行科学记数法转换

步骤： 单精度浮点数转换为IEEE规定的标准编码，需要借助科学计数法。例如将12.25f保存到内存中，需要先转换成对应的二进制数 1100.01 12.25用IEEE编码的显示 用科学记数法表示为 1.10001*2^3(整数部分为1100，小数部分为01，小数点向左移动，每移动一次加一，遇1停止 即为n的算法) n为指数，加上127 转化后用二进制拼接写出 注：为什么要加127呢，十进制127用二进制表示数01111111，IEEE码规定指数域小于01111111为负数，大于为正数，所以01111111为0 由于在转化为二进制的时候舍弃了部分的值，所以C、C++中取得值为近似值，这就解释了浮点数值是否为0时，要做一个区间比较而非等值比较，具体代码如下：

float fTemp =0.0001f
if(fFloat >= -fTemp && fFloat <=fTemp)
{

     //fTemp=0
}
